Wizard is a portfolio HTML template, using most popular design trends as HTML5, CSS3 and jQuery. The template have fullpage scroll function and separate panel section for the homepage, so you can create various styles of homepage to suit your needs, also the template included fully responsive media grid plugin in a portfolio page that allows you to control the layout of your item grid in the way that pleases you most. Wizard is great for creative portfolio website, small business website or any creative business and agency, also have commented HTML code for each panel included in the theme so you can easily adapt your templates to suit your needs.

Features

  • Included Fullpage jQuery Plugin
  • Bootstrap 3 Grid System
  • Homepage Slider Image
  • Homepage Video Background (use background image instead on mobile devices)
  • Vaild HTML5 Pages
  • Responsive Design
  • Retina Ready
  • Responsive Media Grid Plugin
  • Portfolio Filter
  • Working with PHP Contact Form
  • Easy to Customization
  • Well Document
  • Support
  • People and Products photo used for preview only and not included

Files

Inside the "HTML Template" folder you see the following folders.

1. Example Folder

There are six pre-defined styles of the template that you can use. These are:

  • Business
  • Creative
  • Fashion
  • Oganic
  • Personal Portfolio
  • Photography
2. Panels and Pages Template

There are all HTML panels and pages template that you can mix the panels template to create a homepage style to suit your needs.

  • File number 01-34 are panel section that you can combine them to create your homepage.
  • File number 35-42 are single page template.

Create Homepage

To get started you need to copy the necessary resource files and folders below from "Panels and Pages Template" folder and place to your root path.

  • [Folder] bower_components
  • [Folder] css
  • [Folder] fonts
  • [Folder] images
  • [Folder] js
  • [Folder] media
  • [File] mailer.php

Next, create a new HTML file and save as "index.html" to your root path,
and put the CSS link below in the <head> section of your HTML file.

<!--Bootstrap Element and Grid System-->
<link href="css/bootstrap.css" rel="stylesheet" type="text/css">

<!--Main Elements CSS-->
<link href="css/style.css" rel="stylesheet" type="text/css">

<!--Font Awesome-->
<link href="fonts/font-awesome-4.3.0/css/font-awesome.min.css" rel="stylesheet" type="text/css">

<!--OWL Carousel CSS-->

<link href="bower_components/owl.carousel/assets/owl.theme.default.css" rel="stylesheet" type="text/css">

<!--FlexSlider CSS-->
<link href="bower_components/FlexSlider/flexslider.css" rel="stylesheet" type="text/css">

<!--Megafolio CSS-->
<link href="bower_components/megafolio/css/settings.css" rel="stylesheet" type="text/css">

<!--FancyBox CSS-->
<link href="bower_components/fancybox/jquery.fancybox.css?v=2.1.5" rel="stylesheet" type="text/css">

<!--jQuery Fullpage CSS-->
<link href="css/jquery.fullPage.css" rel="stylesheet" type="text/css">

<link href="bower_components/tooltipster/css/tooltipster.css" rel="stylesheet" type="text/css">

Panel Section

The panel section have a HTML markup below.

<div class="section panel-1" data-bg-color="#f7f7f7" data-image-src="images/blank.gif">
	...HTML Code...
    <div class="overlay" data-bg-color="#f7f7f7" data-opacity="0"></div>
</div>

You can change panel background color by edit data-bg-color="" in div class section, and you can insert background image to each panel by edit data-image-src="". The background image by default is data-image-src="images/blank.gif".

<div class="section panel-1" data-bg-color="#f7f7f7" data-image-src="images/blank.gif">

If you already added background image you can add color overlay on image by edit data-bg-color="" in div class overlay, and you can set opacity of color overlay by edit data-opacity="". The opacity by default is data-opacity="0".

<div class="overlay" data-bg-color="#f7f7f7" data-opacity="0"></div>

Video Background Panel

You can change the video background by edit video source in source tag, please prepare your video file in mp4 and webm format.

<!--Video Container-->
<div class="myVideo" data-image-src="images/upload/video-bg-image-small-devices.jpg">
  <video autoplay loop muted controls>
    <source src="media/video/my-video.mp4" type="video/mp4">
    <source src="media/video/my-video.webm" type="video/webm">
  </video>
</div>
<!--End-->
The video background not support on mobile devices and will use image background instead, you can add the image background for mobile devices by edit data-image-src="".

Portfolio Panel 1

How to change layout grid items

You can change layout grid items by edit layoutarray:[] in jquery setting. Find the code below in custom.js file (code line number 488-494).

	// ===== jQuery Megafolio Settings ===== //
	var api=jQuery('.portfolio-panel1 .megafolio-container').megafoliopro({
		filterChangeAnimation: "pagemiddle",
		filterChangeSpeed: 800,
		paddingHorizontal: 0,
		paddingVertical: 0,
		layoutarray: [11]
	});

There are setting for grid style.

layoutarray:[10] for basic 3 columns
layoutarray:[11] for basic 4 columns
layoutarray:[12] for basic 5 columns
layoutarray:[7] for special array 7
layoutarray:[8] for special array 8
layoutarray:[9] for special array 9
layoutarray:[15] for different height 1
layoutarray:[16] for different height 2
layoutarray:[0] for random style

For more layout grid items, please read in this document file.

How to change item hover color

The item hover color by default is #00acc1 that you can change it to suit your needs by edit code line number 148 in color.css file. Also, you can change other element color of portfolio (e.g. category filter hover, link icon hover, related work thumbnail hover in portfolio single page) by edit code line number 128-154 in color.css file.

var customScroll = false;

destroyScroll.slimScroll({

buildScroll.slimScroll({
